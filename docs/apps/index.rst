Following TF decomposition, it can be helpful to visualize the TF data as surfaces or scalp topographic plots. For single-trial data this can help visually identify remaining artifacts in the data that should be deleted, and for all TF transformations this can help clarify scalp distributions and temporal characteristics of power.
